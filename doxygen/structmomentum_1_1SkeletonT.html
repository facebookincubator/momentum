template<typename T >
bool momentum::SkeletonT< T >::isAncestor ( size_t  jointId,
size_t  ancestorJointId 
) const

Check whether the two input joints lie on the same branch of the hierarchy.

Returns true if ancestorJointId is an ancestor of jointId; that is, if jointId is in the tree rooted at ancestorJointId. Note that a joint is considered to be its own ancestor; that is, isAncestor(id, id) returns true.
